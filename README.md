# EchoMate-SelfVoice-VI

Dự án self-bot Discord nói chuyện bằng giọng nói, thuần tiếng Việt.

- Sử dụng discord.js-selfbot-v13
- Bộ não trả lời: Gemini API
- Không train model
- Phù hợp máy yếu

⚠️ Lưu ý: Dự án sử dụng self-bot, có rủi ro vi phạm ToS Discord.
Chỉ dùng cho mục đích học tập và thử nghiệm.
